# How to Use the 「Quantum Open Development Ecosystem Optimization Division Specification Document」

## For Readers (Reference Only)

If you only need to read or refer to the specifications, you can access the latest versions via the GitHub Artifacts or the Releases page.

## For Contributors

To contribute to the specification document, please follow these steps:

1. Join the Q-STAR Alliance: Ensure you/your organization are registered members.

2. Create an Issue & Branch: Open a GitHub issue describing your proposed changes, then create a new branch from main.

3. Submit a Merge Request: Once your changes are ready, submit a Merge Request (MR) for review.

4. Request a Merge: Notify the maintainers to review and merge your branch.

<small>**Note:** When your branch is merged into the main branch, a GitHub Workflow will automatically convert the Markdown files into PDF and HTML formats.</small>

5. Download the Artifacts: You can then download the updated specification document set from the GitHub Actions artifact section.
